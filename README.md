# 纯前端去水印工具 (Pure Frontend Watermark Remover)

[简体中文](./README.md) | [English](./README_en.md)

一个强大且保护隐私的去水印工具，完全基于 Web 技术构建。本应用直接在您的浏览器中运行高度优化的 **基于样本的图像修复 (Criminisi)** 算法，无需后端服务器、云端处理或 API 调用。

## ✨ 特性 (Features)

- **100% 本地处理：** 所有图像处理均通过 HTML5 Canvas API 和 JavaScript `Uint8Array` 在您的浏览器中完成。您的图片绝不会离开您的设备，确保绝对的隐私安全。
- **高级图像修复算法：** 采用现代化的 Criminisi 算法实现，包含：
  - **基于图块的纹理合成：** 重建复杂的背景（如草地、砖墙、纹理），而不仅仅是模糊处理。
  - **结构保留：** 智能优先连接断裂的线条和边缘（数据项与置信度项）。
  - **空间惩罚与 Alpha 混合：** 防止大面积区域变成单一的扁平颜色，并消除图块合并时的生硬接缝。
  - **动态图块大小：** 根据水印区域自动调整处理块的大小。
- **本地图库：** 自动将处理后的图片保存到浏览器的 IndexedDB 中，方便日后查看。
- **现代化 UI：** 使用 React 和 Tailwind CSS 构建，提供简洁、响应式的拖拽体验。

## 🛠 技术栈 (Tech Stack)

- **框架：** React 18 + TypeScript + Vite
- **样式：** Tailwind CSS
- **图标：** Lucide React
- **存储：** IndexedDB (浏览器本地存储)

## 🚀 如何部署 (How to Deploy)

由于这是一个**纯前端应用**（单页应用 SPA），部署极其简单。没有需要维护的 Node.js 后端服务器；您只需要托管静态文件即可。

### 前置要求
- Node.js (推荐 v18 或更高版本)
- npm (或 yarn/pnpm)

### 本地开发
1. 克隆仓库并进入项目文件夹。
2. 安装依赖：
   ```bash
   npm install
   ```
3. 启动开发服务器：
   ```bash
   npm run dev
   ```
4. 打开浏览器并访问终端中提供的 URL（通常是 `http://localhost:3000` 或 `http://localhost:5173`）。

### 生产环境构建
生成可用于生产环境的静态文件：
```bash
npm run build
```
优化后的 HTML、CSS 和 JS 文件将生成在 `dist/` 目录中。

### 部署选项
您可以将 `dist/` 文件夹的内容部署到任何静态托管服务：

- **Vercel / Netlify / Cloudflare Pages：** 直接连接您的 Git 仓库。将构建命令设置为 `npm run build`，输出目录设置为 `dist`。
- **Nginx / Apache / 传统 Web 托管：** 将 `dist/` 文件夹的内容直接复制到 Web 服务器的公共根目录（例如 `/var/www/html`）。
- **GitHub Pages：** 使用 `gh-pages` npm 包或 GitHub Actions 将 `dist/` 文件夹部署到您的 `gh-pages` 分支。

## 📖 如何使用 (前端指南)

本应用的使用非常直观，最终用户无需任何技术背景。

1. **上传图片：**
   - 点击编辑器顶部的 **"Upload Image" (上传图片)** 按钮。
   - 或者，直接从您的电脑将图片文件**拖拽**到灰色的大画布区域。

2. **框选水印：**
   - 图片加载到画布后，您的鼠标光标会变成十字准星。
   - **点击并拖动**鼠标，在水印、文字或不需要的物体上画一个红色的选择框。
   - *提示：尽量让选择框贴近水印边缘。选择区域越小，处理速度越快，结果也越准确。*

3. **去除水印：**
   - 点击紫色的 **"Remove Watermark" (去除水印)** 按钮。
   - 浏览器将开始进行密集的数学计算。由于是本地运行，较大的选择区域或超高分辨率的图片可能需要几秒钟。按钮会显示 "Processing..." (处理中) 动画。
   - 完成后，水印将消失，并被合成的背景纹理所取代。

4. **下载与保存：**
   - 如果您对结果满意，点击绿色的 **"Download & Save" (下载并保存)** 按钮。
   - 这将同时执行两个操作：
     1. 触发标准的浏览器下载，将清理后的 `.png` 图片保存到您的电脑。
     2. 将图片保存到浏览器的本地数据库 (IndexedDB) 中。

5. **管理本地图库：**
   - 查看右侧的 **"Local Gallery" (本地图库)** 侧边栏。
   - 这里会显示您在此浏览器上处理并保存的所有图片的历史记录。
   - 鼠标悬停在图库中的任何图片上，即可显示快捷操作：
     - **下载图标：** 重新将图片下载到电脑。
     - **垃圾桶图标：** 从浏览器的本地存储中永久删除该图片。
   - *注意：由于使用的是 IndexedDB，清除浏览器的网站数据/缓存将会清空此图库。*
